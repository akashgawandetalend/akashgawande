### Method 1: GitHub Pages (Recommended - FREE)

**5-Minute Setup:**

1. **Create Account**: Go to github.com and create a free account
2. **New Repository**: Click "+" â†’ "New repository"
3. **Name It**: `yourusername.github.io` (use YOUR GitHub username)
4. **Upload Files**:

   - Drag ALL files from the portfolio folder
   - Click "Commit changes"
5. **Enable Pages**:
   - Go to Settings â†’ Pages
   - Source: main branch
   - Save
6. **Done!** Your site is live at `https://yourusername.github.io`

If your GitHub username is "akashgawande", create repository: `akashgawande.github.io`

### Method 2: Netlify (Super Easy - FREE)

**3-Minute Setup:**

1. Go to [netlify.com](https://www.netlify.com/)

4. Done! You get a URL like: `yourname-portfolio.netlify.app`
